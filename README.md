# DarkTriad

DarkTriad is a private-use Minecraft Fabric client mod for 1.21.10.

## Requirements
- Minecraft 1.21.10
- Java 21
- Fabric Loader 0.18.4+
- Fabric API 0.138.4+1.21.10

## Install (Client)
1) Install Fabric Loader for 1.21.10.
2) Download Fabric API `0.138.4+1.21.10`.
3) Download `darktriad-1.0.0.jar` from this repository.
4) Place both jars into your `.minecraft/mods` folder.
5) Launch Minecraft using the Fabric profile.

## Default Keybinds
- Open menu: `=`
- X-ray: `X`
- KillAura: `C`
- BowAimbot: `V`
- AutoFarm: `U`
- AutoTree: `I`

Keybinds are configurable in the in-game menu.

## Features (Client-Side)
- X-ray (Sodium compatible)
- Fullbright
- KillAura (range + target filters)
- BowAimbot (prediction + smoothing)
- AutoFarm, AutoTree, AutoTotem
- ESP (players, mobs, items)
- QuickBreak, NoFall, NoWeather, NoFireOverlay
- HUD toggles and status list

## Notes
- This mod is intended for private servers with consent.
- Config file: `config/darktriad.json`
